INTRODUCTION AU HTML5

HTML est l'abreviation de Hyper Text Markup Language, qui signifit en français "Langage HyperTexte de Balisage". Ce langage a été créer en 1991 et a pour fonction de donner du sens a un contenu.
Le CSS qui veut dire Cascading Style Sheets signifit en français "Feuille de Style en Cascade" a été créer en 1996. Ce langage a pour role de mettre en forme un contenu tout en appliquant du style a ce dit contenu.

Pourquoi apprendre les langages HTMl et CSS?

Tout d'abord, il faut savoir que le HTML et le CSS sont deux comme cela peut etre le cas pour le langage PhP par exemple. Si le monde de l'informatique vous plait et que vous etes tenté par un projet de developpement Web, que vous voulez créer un veritable site standards et dynamique, l'apprentissage des langage HTML et CSS sont primordiale voir meme basique pour toi car le HTML et le CSS sont de veritable socle pour la creation des pages Web jusqu'aux applications mobile en passant par les blogs voir meme des sites e-commerce en integralité, le plus important dans tout ça est que le HTML et le CSS constitut les bases de la programmation. donc pour devenir un bon programmeur l'apprentissage du HTML et du CSS est d'une importance capitale pour toi.

il est a noter que le HTML n'est pas un langage de programmation mais plutot un langage de "balisage".

Les fondations du HTML5

Elements, Balises et Attributs

Les elements vont nous servir a definir des objets dans le HTML.
Ce sont les elements balises et attributs qui vont nous permettre de donner la forme a notre pages Web, les element sont constitués de balises. Dans la majorité des cas, il y a trois termes que vous devez comprendre le sens qui sont a savoir :Les titres, les paragraphes et les balises.
